Установка Yii2 (шаблон basic) в Ubuntu

  • 1. Перед установкой Yii2 проекта необходимо:
    • 1.1 Установить PHP 7
    • 1.2 Установить composer
  • 2. Установка проекта Yii2
    • 2.1 Установка проекта в папку basic:
      • composer create-project --prefer-dist yiisoft/yii2-app-basic basic
    • 2.2 Переходим в папку проекта:
      • cd basic
    • 2.3 Запускаем встроенный в PHP веб сервер:
      • php -S localhost:3000
    • 2.4 Переходим на страницу проекта:
      • localhost:3000/web/index.php
    • 2.5 Переходим на страницу с чек листом:
      • localhost:3000/requirements.php
    • 2.6 Производим установку недостающих пакетов согласно чек листу.
