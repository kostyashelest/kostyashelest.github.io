Установка и настройка LAMP в Ubutnu

  • 1. Обновление системы:
    • sudo apt update
    sudo apt upgrade
  • 2. Установка Apache2
    • sudo apt install apache2
  • 3. Установка MySQL
    • sudo apt install mysql-server
  • 4. Устанавливаем PHP
    • sudo apt install php7.4 libapache2-mod-php7.4 php-mysql
  • 5. Ставим расширения для PHP
    • sudo apt install php-curl php-json php-cgi php-gd php-zip php-mbstring php-xml php-xmlrpc
  • Примечание:
    • После смены директории с сайтами возникает ошибка 403. Необходимо сменить корневую директорию в файле: sudo nano /etc/apache2/apache2.conf А также, в конфиге хоста.
